Wp Ultimate Review

== Description ==

This review plugin helps you to interact with your users with improved your blog comments. visitors submit reviews for different posts products, events or anything can write reviews. you can expose your reviews in a easier to read way and will help you generate money by affiliates adding a buy now button in your post.



###SUPPORT###

Found issue or new features? [Contact our team from here](https://help.wpmet.com/).



You can set global product review also users can review by self in your post.




==== KEY FEATURES ====
- Set global product review in your blog.
- Accept and manage user reviews
- Multiple rating style, like Point, star, Percents and pie chart
- Allow create additional review fields, like value, appearance, quality, etc.
- Set the maximum score for your product review and use either a text input, dropdown selection or number of stars for score input
